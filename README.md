BlueFuture — Group Website Coursework (4COSC011W)

A multi-page website built as a group coursework project for module 4COSC011W, 
based on the assigned theme "Life Under Water" (Theme 14). 
The site was developed collaboratively, with each team member responsible for 
individual pages while sharing a common template, navigation, and styling.

[Overview]
This project is a static website consisting of a shared splash screen, home page
and several feature pages, all built on a consistent global template (shared header, 
footer and navigation bar)and styled with a common CSS theme inspired by underwater/marine 
life visuals (ocean colour palette, sea creatures, coral, etc.).

[Pages / Features]

Splash Screen – Landing/intro screen shown before the main site loads
Home Page – Main landing page with site overview and navigation
Gallery – Image gallery showcasing underwater life and marine visuals
AIS (About / Information / Services — update to match your actual page)
Feedback – Page for collecting user feedback (e.g. a form)
Team – Information about the project team members
User Profile – User profile page
Sitemap – SVG-based visual sitemap of the website


[Tech Stack]
HTML5 – Page structure and content
CSS – Global styling and layout (shared template/theme across pages)
JavaScript – Interactivity and dynamic behaviour
